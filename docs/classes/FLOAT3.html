Summary

96-bit wide struct holding three floats, named _x, _y, and _z. This class has no methods. Located in FloatStructs.h, under the namespace Frost.

The definition, as found in FloatStructs.h, is:

struct FLOAT3 { float _x; float _y; float _z; };

Properties:

_x First float in the structure (x component)
_y Second float in the structure (y component)
_z Third float in the structure (z component)

Methods:

The FLOAT3 struct has no methods.

Usage Notes:

The FLOAT3 structure was built to make derived classes play nicely with each other, as well as external structures like XMMath's XMFLOAT3 and Direct3D's D3DXVECTOR3. Any other 96-bit float3 structure can be (I don't know if I'd reccomend this) casted to a FLOAT3 as such:

FLOAT3 myFloat = *reinterpret_cast<FLOAT3*>(&other_3_float_var);
