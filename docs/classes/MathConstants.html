Indigo Frost Math Constants List

Summary

Indigo Frost, by nature, is very mathematical. As such, certain mathematical constants have been defined for use in development, testing, and implementation.

All of these mathematical constants are found in FrostMath.h. Conviniently, FrostMath.h also includes all of the files in the math tier of the engine, so including FrostMath.h in your project gives you access to all of the mathematical classes that the engine uses.

All of these constants are found in the Frost::MathConstants namespace.

Scalar Constants

ConstantValueNotes
PI 3.141592654f
PI_DIV_2 1.5707963265f
PI_DIV_3 1.047197551f
PI_DIV_4 0.78539816325f
PI_MULT_2 6.283185306f
DEG_TO_RAD PI / 180.f Multiply an angle in degrees to get the value in radians.
RAD_TO_DEG 1.f / DEG_TO_RAD Multiply a value in radians to get the angle in degrees.

VECTOR_UNIT_Y <0.f, 1.f, 0.f>

QUATERNION_UNIT 1.f (+ 0i + 0j + 0k) This isn't a Vect3, but it doesn't belong anywhere else... Unit quaternion, representing a zero rotation.
