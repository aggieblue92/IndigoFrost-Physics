Summary

Vect3 holds data for a mathematical vector in 3D space. Being inherited from FLOAT3, it plays nicely with other 3D vector data types used in other libraries, like XMFLOAT3 and D3DXVECTOR3.

Vect3 holds methods for performing regular vector mathematics functions, like the dot (or scalar) product, cross (or vector) product, computing the magnitude of the vector, the square magnitude (which is faster), etc.

Vect3 is held in Vect3.h

Properties:

_x X component of the vector. Derived from FLOAT3
_y Y component of the vector. Derived from FLOAT3
_z Z component of the vector. Derived from FLOAT3

Add by component a 3D float structure to the Vect3 instance, return the sum.

Returns a new vector, representing the sum of the two vectors.

other The FLOAT3 type (Vect3, Vect3Normal, FLOAT3, etc.) to add to the vector.

Vect3 operator-(const FLOAT3& other) const

Subtracts by component a 3D float structure to the Vect3 instance, return the difference.

Returns a new vector, representing the difference of the two vectors.

other The FLOAT3 type (Vect3, Vect3Normal, FLOAT3, etc.) to subtract from the vector.

float operator*(const FLOAT3& other) const

Performs a scalar product (aka dot product) on the two vectors, returning the result. Mathematically, this is scalar_product = x1*x2 + y1*y2 + z1*z2

Returns: The scalar product of the two vectors

other The other vector involved in the operaton.

Vect3 operator%(const FLOAT3& other) const

Performs a vector product (aka cross product) on the two vectors, returning the result. The resultant vector will be orthogonal to both vectors involved in the operation. This is frequently used in computing torque. This cross product assumes the right hand rule.

Returns: The vector product of the two vectors

other The other vector involved in the operaton.

Vect3 operator*(float other) const

Multiplies the vector by a scalar value. Every component of the vector is multiplied by said scalar.

Returns: The instance vector scaled by the value other

other The scale to apply to the instance vector.

Vect3 operator/(float other) const

Multiplies the vector by the inverse of a scalar value. Every component of the vector is multiplied by said scalar inverse.

Returns: The instance vector scaled by the value 1.0f/other

other The scale to invert and to apply to the instance vector.

Vect3& operator+=(const FLOAT3& other)

Adds together the instance vector and the vector specified in other, modifying the instance vector and returning it by reference.

Returns: A reference to the modified instance vector.

other The vector which is to be added, by component, to the instance vector.

Vect3& operator-=(const FLOAT3& other)

Subtracts the vector specified in other from the instance vector, modifying the instance vector and returning it by reference.

Returns: A reference to the modified instance vector.

other The vector which is to be subtracted, by component, from the instance vector.

Vect3& operator*=(float other)

Multiplies every component in the instance vector by the value specified in other, and then returns a reference to the instance vector.

Returns: A reference to the modified instance vector.

other The scalar by which to multiply the components of the instance vector.

Vect3& operator/=(float other)

Divides every component in the instance vector by the value specified in other, and then returns a reference to the instance vector.

Returns: A reference to the modified instance vector.

other The scalar by which to divide the components of the instance vector.

float operator[](int i) const

Returns by value the component at index i of the vector (x=0, y=1, z=2)

Returns: The specified component of the vector

Throws IndexOutOfBoundsException if 0<=i<=2 is not met.

i 0, 1, or 2, corresponding to x, y, and z, respectively.

float& operator[](int i)

Returns by reference the component at index i of the vector (x=0, y=1, z=2). This allows modification of the vector, and is the default version of this function to be called.

Returns: The specified component of the vector

Throws IndexOutOfBoundsException if 0<=i<=2 is not met.

i 0, 1, or 2, corresponding to x, y, and z, respectively.

Vect3& operator=(const FLOAT3& other)

Assigns the value in other to the instance vector, and then returns by reference the instance vector.

Returns: A reference to the finished instance vector.

other The vector which is to be copied into the instance vector.

bool operator==(const FLOAT3& other)

Compares the value in the instance vector to that specified in the other vector.

Returns: true if the two vectors have equivalent components, false otherwise.

other The vector to compare, by component, to the instance vector.

bool operator!=(const FLOAT3& other)

Compares the value in the instance vector to that specified in the other vector.

Returns: false if the two vectors have equivalent components, true otherwise.

other The vector to compare, by component, to the instance vector.

float Magnitude() const

Returns the magnitude of the instance vector, but runs slower than SquareMagnitude. When possible, use SquareMagnitude instead (for comparing the magnitude of two vectors, for instance)

Returns: the magnitude of the instance vector.

float SquareMagnitude() const

Runs faster than Magnitude

Returns: the squared magnitude of the instance vector.

Usage Notes:

I personally like to construct this object like so:

Frost::Vect3 myVect({x, y, z});

Also, there are pre-defined useful vectors under the MathHelpers namespace:

  • Frost::MathHelpers::VECTOR_UNIT_X
  • Frost::MathHelpers::VECTOR_UNIT_Y
  • Frost::MathHelpers::VECTOR_UNIT_Z
  • Frost::MathHelpers::VECTOR_ZERO

If you would rather perform vector functions outside of an instance of a vector, they are provided in functions in the Frost namespace. Additionally, a ComponentProduct function is added, which performs per-component multiplication of two vectors. You may find that useful.

  • float DotProduct(const Vect3& v1, const Vect3& v2);
  • Vect3 CrossProduct(const Vect3& v1, const Vect3& v2);
  • Vect3 ComponentProduct(const Vect3& v1, const Vect3& v2);
