Summary

Vect3Normal is simply an extension of Vect3, with the only difference being that it is guaranteed to be a normal vector (have a magnitude of 1.0). Any time a function is called that can modify the Vect3Normal instance, the normalization process is performed. So, any time you use it in your code, you are guaranteed a normal vector. This is especially useful in vector mathematics.

Vect3Normal is held in Vect3Normal.h
